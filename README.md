# Structura

**Professional calculation tools for architects and engineers.**

🌐 Live site: [architecture-tool-kits.vercel.app](https://architecture-tool-kits.vercel.app)

---

## About

Structura is a free web-based toolkit that brings the most-used architectural and engineering calculations into one clean, fast interface. No login, no setup, no noise.

Built to solve a real problem — architects and engineers waste significant time on repetitive calculations done on scraps of paper, in Excel, or with scattered online tools that are slow and unreliable.

---

## Tools

| Tool | Description | Standard |
|------|-------------|----------|
| Scale Converter | Real dimensions → model scale | — |
| Stair Calculator | Step count and stair geometry | IBC 2021, EN 13101 |
| Ramp Calculator | Slope and accessibility compliance | ADA, EN 17210 |
| Column & Beam Span Calculator | Structural span and slab depth | Eurocode 2, ACI 318 |
| Site Coverage Calculator | FAR, plot ratio, floor area | Local zoning |
| Parking Calculator | Vehicle capacity and layout | ITE, AS/NZS 2890.1 |
| Room Program | Space schedule and area builder | Neufert, ISO 9836 |
| Fire Escape Calculator | Exit distance and evacuation | IBC 2021 |
| Daylight Calculator | Natural light compliance | EN 17037, IES |
| Wall U-Value Calculator | Thermal transmittance | ASHRAE 90.1, Eurocode |
| Structural Grid Calculator | Column grid and bay planner | Eurocode 2, ACI 318 |
| Building Load Calculator | Floor load and weight estimator | Eurocode EN 1991, ASCE 7 |

---

## Features

- 12 professional calculators in one place
- EN / TR language support
- Dark / light mode
- PDF and CSV export for all tools
- Live SVG diagrams that update with inputs
- International standards references (IBC, Eurocode, ADA, ASHRAE)
- Mobile responsive
- No login required — completely free

---

## Tech Stack

- **React** — UI framework
- **Vite** — build tool
- **Tailwind CSS** — styling
- **GSAP** — animations
- **React Router v6** — navigation
- **jsPDF** — PDF export
- **Vercel** — deployment
